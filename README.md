## :toolbox: Getting Started

### :gear: Installation

To Run
```bash
Open E Clock.exe file
```


### :running: Run Locally

Clone the project

```bash
https://github.com/BhukuDon/E-Clock-Timer
```
Install dependencies
```bash
pip install -r requirements.txt
```
Build
```bash
python3 builder/builder.py
```
